Graphing Calculator - Clues

Symbols: + - * / ^

The usual precedence.

3x == 3 * x
10 sin x == 10 * sin(x)
sin(10x) == sin(10 * x) ... however
sin 10x == sin(10) * x

Keywords to try:

Unary operators:

  • /x == 1/x
  • *x == sgn(x)
  • ^x == e^x
